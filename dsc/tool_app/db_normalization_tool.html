🔍 属性闭包计算
给定属性集 A，计算 A 在函数依赖集 F 下的闭包 A⁺——即 A 能函数决定的所有属性集合。
// 属性闭包算法
result ← A
while(result 有变化) {
  for_each f ∈ F:
    if(f.left ⊆ result):
      result = result ∪ f.right
}
return result
✂️ 正则覆盖（Canonical Cover）
去除函数依赖集 F 中的冗余，得到等价的最小覆盖 Fc，满足：无无关属性、无冗余FD、右侧为单属性。
⚖️ BCNF 判定
检验模式 R 在 F 上是否满足 BCNF：对所有非平凡 α→β ∈ F⁺，α 必须是 R 的超键。
